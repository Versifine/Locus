### R06: 混合推理架构（Hybrid Brain）

**状态**: 探索中  
**核心问题**: 如何设计一个成本可控、仿生的多层 AI 架构？

| 文档 | 内容 |
|------|------|
| [2026-02-03 讨论日志](logs/2026-02-03-hybrid-brain-architecture.md) | 初次讨论：本地模型 vs API、仿生架构、三层设计 |

**关键结论**:
- 当前多模型架构是"串行狭窄高速公路"，与生物大脑的"广泛并行连接"相反
- 提出三层仿生架构：Reflex(规则) → Fast(本地模型) → Slow(云端API)
- 需要 Blackboard 共享状态机制，而非串行消息传递

---

### R07: 肌肉记忆学习

**状态**: 待研究  
**核心问题**: 如何让 Agent 像人类一样学习和优化动作技能？

**学习路径**: 预定义 → 模仿学习 → 强化学习 → 自主创造

**相关领域**: 运动学习、强化学习、Hierarchical RL、程序性记忆

## 如何使用此目录

1. **PO 模式讨论后**：将结论写入对应日志，更新本索引
2. **新研究主题**：在此 README 新增条目，创建对应日志文件
3. **论文阅读后**：在 `papers/` 创建阅读笔记，更新阅读计划状态
4. **结论成熟后**：迁移到 PRD 成为正式需求
